### GameArchitecture

An implementation of ScriptableObject as values / events as per the Unite Austin 2017 talk [Game Architecture with Scriptable Objects](https://www.youtube.com/watch?v=raQ3iHhE_Kk).

#### Features:
* Event
* EventListener
* Value
* Reference
* Set

#### Supported types:
* void (Events/Listeners only)
* int
* float
* bool
* string
* Vector2
* Vector3
* Color
* Transform
* GameObject
